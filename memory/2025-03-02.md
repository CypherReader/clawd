# 2025-03-02 - Cross-Platform Arbitrage Research

## Phase 1: Platform Research Completed

### Key Platforms Analyzed:
1. **Kalshi (CFTC-Regulated)**
   - Official REST API with public market data access
   - Binary markets, prices in USD cents
   - No authentication needed for data retrieval
   - Clean JSON structure, well-documented

2. **Azuro (Protocol Layer)**
   - "Uniswap for betting" - liquidity pool model
   - Multi-chain: Polygon, Gnosis, Base, Chiliz
   - Complex architecture: GraphQL subgraphs + REST + WebSocket
   - Protocol layer powers multiple frontends

3. **PredictIt (Politics-Focused)**
   - Unofficial/community reverse-engineered API
   - XML/CSV data format (not JSON)
   - Political market specialization
   - $850 maximum investment per contract
   - Academic/research focus with non-commercial terms

### Strategic Insights:
- **Kalshi**: Best starting point - low risk, official API
- **PredictIt**: Unique political arbitrage opportunities
- **Azuro**: Most complex but innovative protocol-layer arbitrage

### Arbitrage Opportunities Identified:
1. **Regulatory Arbitrage**: Kalshi (regulated) vs less-regulated platforms
2. **Specialization Arbitrage**: PredictIt politics vs general platforms  
3. **Architectural Arbitrage**: Azuro liquidity pools vs order books
4. **Cross-Chain Opportunities**: Azuro multi-chain vs single-chain
